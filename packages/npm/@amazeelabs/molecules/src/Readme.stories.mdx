# Amazee Labs Molecules

A collection of reusable React components, built to be styled using [tailwindcss](https://tailwindcss.com).

 ## Installation

```shell
npm install @amazeelabs/molecules
```

## Usage

Simply import and use components. They are optimized and tree-shakeable.
